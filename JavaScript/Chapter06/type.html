Symbol : 코드 내에서 유일한 값을 가진 변수 이름을 만들 때 사용
다른 어떤 값과 비교해도 true가 될 수 없는 고유한 변수가 됨
Ex)  const user1 = Symbol('this is a user');
      const user2 = Symbol('this is a user');
      console.log(user1 === user2) //false

BigInt : 아주 큰 정수를 표현하기 위한 데이터 타입

typeof 연산자 : 공백을 두고 작성해도 되고, 함수처럼 괄호로 감싸서 사용해도 됨
typeof null; // object
typeof 함수명; // function
